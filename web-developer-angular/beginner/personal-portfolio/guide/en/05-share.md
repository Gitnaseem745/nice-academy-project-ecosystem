This is the easiest way to host a static site for free.

1.  **Push to GitHub**: Make sure your code is in a repository on GitHub.
2.  **Settings**: Go to your Repo -> Settings -> Pages.
3.  **Source**: Select "Deploy from a branch".
4.  **Branch**: Select `main` (or `master`) and `/root` folder.
5.  **Save**: Wait a minute, and you will get a live URL!
