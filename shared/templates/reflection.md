# Project Reflection

Complete this after finishing the project and sharing it publicly.

This reflection is for **learning clarity**, not grading.

---

## 1. What Did You Build?

Explain in your own words what this project is and what it does.

---

## 2. What Was Harder Than Expected?

Be specific.
Avoid generic answers.

---

## 3. What Did You Learn?

List:
- Technical learnings
- Design decisions
- Process insights

---

## 4. What Would You Improve Next?

Explain what you would do differently if you rebuilt this project.

---

## 5. Confidence Check

Answer honestly:
- Can you explain this project without notes?
- Can you rebuild something similar from scratch?

If not, identify the gap.
